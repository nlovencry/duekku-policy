1

Pendahuluan

Selamat datang di Duekku ("Aplikasi"). Kami sangat menghargai kepercayaan Anda dan berkomitmen penuh untuk melindungi privasi serta data pribadi Anda sebagai pengguna. Kebijakan Privasi ini dirancang untuk menjelaskan bagaimana kami mengumpulkan, menggunakan, menyimpan, dan melindungi informasi Anda saat menggunakan aplikasi pelacak keuangan pribadi Duekku.

Dengan mengunduh, mendaftar, atau menggunakan Aplikasi ini, Anda dianggap telah membaca, memahami, dan menyetujui seluruh ketentuan yang tercantum dalam Kebijakan Privasi ini.

2

Data yang Kami Kumpulkan

Kami mengumpulkan data tertentu untuk mempermudah Anda dalam mengelola keuangan pribadi secara sinkron dan real-time:

A. Informasi Profil Pengguna

Ketika Anda membuat akun, kami mengumpulkan informasi identitas dasar berupa:

  • Nama Lengkap / Nama Panggilan: Untuk personalisasi antarmuka aplikasi.
  • Alamat Email: Digunakan untuk kredensial login (autentikasi), memverifikasi akun Anda, serta memulihkan kata sandi jika diperlukan.
  • Foto Profil: Opsional, jika Anda memilih untuk mengunggah foto profil Anda sendiri.

B. Data Keuangan dan Transaksi

Sebagai inti dari fungsi pelacakan keuangan, Aplikasi menyimpan data keuangan yang Anda masukkan secara manual, mencakup:

  • Kantong Keuangan (Pockets): Nama kantong/dompet (misal: "Dompet Utama", "Tabungan", "E-Wallet").
  • Detail Transaksi: Catatan pemasukan, pengeluaran, transfer saldo antar kantong, nilai nominal transaksi, kategori transaksi, deskripsi/catatan, serta tanggal/waktu transaksi.

C. Berkas Media (Kamera & Galeri)

Aplikasi ini memiliki fitur melampirkan bukti transaksi. Kami akan meminta izin akses ke:

  • Kamera & Galeri Foto: Hanya digunakan untuk mengambil gambar bukti transaksi (nota/struk) atau foto profil baru untuk diunggah ke penyimpanan cloud Anda sendiri.

D. Informasi Teknis & Perangkat

Untuk memastikan stabilitas aplikasi dan membantu proses debugging apabila terjadi kendala teknis, kami mengumpulkan:

  • Model perangkat seluler, versi sistem operasi (Android/iOS), dan tanda pengenal perangkat unik (Device ID).
3

Bagaimana Kami Menggunakan Informasi Anda

Informasi yang kami kumpulkan digunakan secara eksklusif untuk tujuan-tujuan berikut:

  • Penyediaan Layanan Utama: Mengelola kantong keuangan Anda, melakukan kalkulasi saldo, dan menampilkan grafik riwayat transaksi Anda secara akurat.
  • Sinkronisasi Cloud secara Real-time: Menyimpan data di server cloud agar Anda dapat mengakses catatan keuangan Anda secara sinkron dari berbagai perangkat menggunakan akun yang sama.
  • Pembuatan Laporan Keuangan: Memungkinkan Anda mengekspor data transaksi ke dalam dokumen berformat Excel (.xlsx) atau file cetak PDF.
  • Pengiriman Notifikasi (Push Notifications): Memberikan pengingat pencatatan keuangan harian atau informasi penting terkait akun Anda.
  • Peningkatan Kualitas Aplikasi: Menganalisis log error untuk memperbaiki bug/kerusakan demi meningkatkan performa Aplikasi.
4

Layanan Pihak Ketiga

Kami menggunakan infrastruktur cloud backend dari pihak ketiga yang tepercaya untuk mendukung fungsionalitas aplikasi:

  • Google Firebase Services:
    • Firebase Authentication: Mengelola pendaftaran akun baru, login aman, dan reset password.
    • Cloud Firestore: Database terenkripsi untuk menyimpan data transaksi, kantong, dan konfigurasi pengguna.
    • Firebase Storage: Tempat penyimpanan file media bukti transaksi (nota/struk) dan foto profil yang Anda unggah secara aman.
    • Firebase Cloud Messaging (FCM): Mengirim notifikasi lokal maupun push notification ke perangkat Anda.

Kami tidak menjual, memperdagangkan, atau membagikan data transaksi pribadi Anda kepada pengiklan atau pihak ketiga lainnya untuk tujuan pemasaran.

5

Keamanan & Penyimpanan Data

Keamanan data keuangan Anda adalah prioritas kami. Semua data transaksi dan data pribadi disimpan di server cloud Google Firebase yang memiliki enkripsi transmisi data (SSL/TLS) dan enkripsi saat istirahat (encryption at rest).

Meskipun demikian, perlu diingat bahwa tidak ada metode transmisi melalui internet atau metode penyimpanan elektronik yang 100% aman. Kami terus berusaha menggunakan cara-cara yang diterima secara komersial untuk melindungi informasi pribadi Anda, tetapi kami tidak dapat menjamin keamanan mutlak.

6

Hak Pengguna & Penghapusan Akun

Sebagai pengguna, Anda memiliki kontrol penuh atas data Anda sendiri:

  • Akses & Edit Data: Anda dapat melihat, mengedit, atau menghapus setiap catatan transaksi dan kantong keuangan secara langsung di dalam aplikasi kapan saja.
  • Penghapusan Akun & Data Permanen: Jika Anda tidak ingin lagi menggunakan aplikasi, Anda dapat menghapus akun Anda. Anda dapat melakukan penghapusan akun secara mandiri melalui menu Pengaturan Profil di dalam aplikasi, atau mengajukan permintaan penghapusan melalui email ke tim dukungan kami. Panduan langkah-langkah detail mengenai proses penghapusan data ini dapat dilihat di Halaman Panduan Penghapusan Akun. Penghapusan akun akan menghapus seluruh data transaksi, riwayat keuangan, serta file bukti transaksi secara permanen dan tidak dapat dipulihkan.
